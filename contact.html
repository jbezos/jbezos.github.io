Paquetes
Puedes enviar informes de bichos, comentarios y sugerencias sobre los paquetes descritos en este sitio con el siguiente formulario, que son bienvenidos.

Lamento no atender preguntas sobre el uso de estos paquetes (o sobre el uso de TeX o LaTeX), que se pueden dirigir mejor a foros públicos, como la lista de correo es-tex de CervanTEX (que leo con regularidad).

Personal
También puede usar este formulario para mensajes personales. Para dudas sobre lengua y estilo del español, recomiendo acudir al servicio de consultas de la FundéuRAE.

Packages
You can send bug reports, comments and suggestion about the LaTeX packages described in this site, which are welcome, with the following form.

I regret I don’t answer questions about how to use these packages (or about how to use TeX or LaTeX), which are best addressed to public forums, like TeX Stack Exchange (which I read regularly).
